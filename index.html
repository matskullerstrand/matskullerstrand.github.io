Mats Kullerstrand is a UX Designer student at Chalmers University of Technology. He enjoys creating products where the user is in the centrum utilizing programs such as Figma and the Adobe Suite.

Light Rig Application

Graphical User Interfaces

Figma

Tablet App

The purpose of this project was to design a graphical user interface for a complex system, a light rig control system utilized prior to and during concerts. This interface was specifically designed for tablet usage, enabling users to explore their work from different perspectives within the auditorium.

Death Date

Designing User Experiences

Figma

Photoshop

Mobile App

The aim of this project was to create a user experience that evokes the senses of unease and anxiety through critical and speculative design. The outcome, Death Date, is a DNA test that estimates users' natural death dates. Additionally, the company offers an application to help plan your remaining days, including retirement, bucket list activities, and much more.

Knowledge Sharing Platform

Individual Project

User Centered Design

Figma

Desktop App

The aim of this project was to explore how a universal knowledge sharing platform can be developed to suit different corporations' needs. The resulting platform enables coworkers to help each other, ask for help, find experts within the company, and store knowledge from previous projects.

Bachelor Thesis

Product Development

User Centered Design

Design Process

The brief was to investigate how a workplace for a specific operator can be designed when relocating to a new and centeralized controlroom at Vargön Alloys AB. In order to achieve this, the needs and requirements of the mentioned operator were analyzed. This was a group project and was carried out in collaboration with Boid AB who designed the controlroom.
